## To use

- Fork or download this repo

## Setup

Run:

```
yarn install
```

## Development

To start development server

```
make sure convert .env.example. to .env

yarn dev
```

## Deployment

To deploy to vercel now host

```
yarn deploy
```

## Lint code and check Errors

Using Eslint and Prettier

```
yarn lint
```
